\doxysubsection{Detailed Description}
Base class for handling \doxylink{class_m_p_u6050}{MPU6050} sensor data. 

This class provides methods for initializing the \doxylink{class_m_p_u6050}{MPU6050} sensor, reading accelerometer and gyroscope data, calculating angles, and performing sensor calibration and error correction.
