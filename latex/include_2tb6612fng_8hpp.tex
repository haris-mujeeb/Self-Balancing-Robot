\doxysubsection{Detailed Description}
Driver class for the \doxylink{class_t_b6612_f_n_g}{TB6612\+FNG} motor driver, supporting multiple motor control configurations. 

This header file provides an abstraction for controlling motors using the \doxylink{class_t_b6612_f_n_g}{TB6612\+FNG} driver with flexible compile-\/time configuration for different control modes. It supports single-\/direction, inverted pins, and four-\/pin configurations for motor control.

\begin{DoxyNote}{Note}
Ensure that one of the configuration modes ({\ttfamily MOTOR\+\_\+\+MODE\+\_\+\+SINGLE\+\_\+\+DIRECTION}, {\ttfamily MOTOR\+\_\+\+MODE\+\_\+\+INVERTED\+\_\+\+PINS}, or {\ttfamily MOTOR\+\_\+\+MODE\+\_\+\+FOUR\+\_\+\+PINS}) is defined at compile time. 
\end{DoxyNote}
